## 충돌 시 우선순위 (위가 이긴다)
1. **명시적 결정** — `04 §2` 문서 정합성 노트 및 `.claude/rules/decisions.md`(D1~D4). (예: 주관식 정답 단일/콤마 = D1 이 DS-02/DS-03 의 "다중 정답 N개" 묘사를 **무시**)
2. **해당 영역의 관할 문서**(위 표)
3. 그 외 문서
4. **코드 ↔ spec 충돌** → 항상 spec 우선

- spec 에 없거나 서로 모순이고 `04 §2`/decisions 로도 해소 안 되면 → 코드 작성 금지, 사용자에게 질문(🙋🏻).
